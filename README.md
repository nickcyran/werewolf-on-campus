# Godot Project – Setup Guide (Beginner Friendly)

This repository contains a game project built using the **Godot Game Engine**.

You have been invited to this **private repository via email**. Follow the steps below to access the project and run it locally.

## Important Note

Only follow the full setup instructions below **if you plan to contribute code or modify the Godot project locally**.

If you only need to **add or upload documents** (such as PDFs, images, or other files), you can simply use the **GitHub website**:

1. Open the repository in your browser.
2. Click **Add file → Upload files**.
3. Drag and drop your document(s).
4. Commit the changes.

This is often easier and does **not require installing GitHub Desktop or Godot**.

# 1. Accept the GitHub Repository Invitation

1. Check your email for a **GitHub invitation** to collaborate on this repository.
2. Click **Accept Invitation**.
3. Once accepted, the repository will appear in your GitHub account.


# 2. Install GitHub Desktop

GitHub Desktop makes downloading and updating the project much easier.

Download it here:

https://desktop.github.com/

Install and log in using your **GitHub account**.


# 3. Clone the Repository

1. Open **GitHub Desktop**.
2. Click:

```
File → Clone Repository
```

3. Select the **GitHub.com** tab.
4. Find this repository in the list.
5. Choose a location on your computer.
6. Click **Clone**.

GitHub Desktop will download the entire project to your machine.


# 4. Install Godot

Download the Godot engine from the official website:

https://godotengine.org/download

Download the version for your operating system.

Godot does **not require installation**. You simply run the executable.


# 5. Open the Project in Godot

1. Launch **Godot Engine**.
2. In the Project Manager click:

```
Import
```

3. Navigate to the folder where GitHub Desktop cloned the repository.
4. Select the file:

```
project.godot
```

5. Click **Import & Edit**.

The project will now open in the Godot editor.

# 6. Run the Project

To start the game:

Click the **Play button** in the top-right corner of the editor
or press:

```
F5
```

Godot will run the main scene.


# 7. Updating the Project

If changes are pushed to the repository:

1. Open **GitHub Desktop**
2. Click **Fetch Origin**
3. Click **Pull**

Your local project will update to the newest version.

# 8. Troubleshooting

If the project does not run:

- Make sure you accepted the GitHub invitation
- Make sure GitHub Desktop finished cloning the repository
- Verify you imported the correct **project.godot** file
- Confirm you are using **Godot 4.6.1**
